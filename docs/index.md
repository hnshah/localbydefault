# localbydefault

Local-first model orchestration: run local by default, fall back to cloud when needed.

- Domains: localbydefault.com / .ai / .dev
- GitHub: https://github.com/hnshah/localbydefault

## What we're building

An AI-native control plane that lets a cloud model orchestrate a fleet of local models (Ollama / MLX / vLLM), with measurable cost + quality and policy-driven routing.
